 [React Native开发文档](https://reactnative.dev)
> 先配置好安卓开发环境（准备个稳定的翻墙软件），在启动项目。
### For Android
```bash
# using npm
npm i
npm run android
# OR using Yarn
yarn
yarn android
```

### 完成项（android）
1. **应用适配**
2. **自适应主题（深色|浅色）**
3. **开屏封面**
4. **页面鉴权**
5. **部分页面开发**（首页、沸点、发现、课程、我的、登录、搜索、文章详情、沸点详情、webview内嵌、标签管理、阅读记录、内容数据、福利兑换）。
6. **公共组件开发**（顶部导航栏、页面鉴权、搜索框、markdown、评论卡片、文章卡片、沸点卡片、分割线）。
